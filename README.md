# 前言

随着数字音乐的普及，怀旧唱片逐渐淡出了人们的视线。然而，对于那些热爱音乐，喜欢收藏实体唱片的人们来说，一个便于购买和收藏怀旧唱片的平台显得尤为重要。"基于SSM的怀旧唱片销售系统"正是为了满足这部分人群的需求而开发的。本项目使用Java语言，结合Spring、SpringMVC和MyBatis框架，前端采用JS、Vue和CSS3技术，致力于打造一个便捷、实用的怀旧唱片在线销售平台。

# 内容介绍

本项目主要包括用户注册登录、唱片浏览、搜索、购买、评论等功能。用户可以轻松注册和登录，根据自己的喜好浏览不同类型的唱片，还可以通过搜索功能快速找到自己想要的唱片。同时，用户可以在唱片详情页查看唱片的详细信息，包括歌曲列表、封面、价格等，并进行购买和评论。此外，系统还提供了后台管理功能，方便管理员对唱片信息、用户信息等进行管理。

# 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC，MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
